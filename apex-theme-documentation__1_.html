Overview & Features

What's included in the Apex Packaging theme

Apex Packaging is a modern Shopify OS 2.0 theme designed specifically for packaging, B2B product, and food service businesses. It draws inspiration from editorial design with clean typography, smooth animations, and a fully responsive layout.

Key Features

FeatureDescription
Mega MenuFull-width dropdown with auto-pulling collection images, drag-to-scroll, and stagger animations
Collection PageSidebar filters, search, sort, active pills, category tabs, inline delivery & rewards banners, load more
Product PageSticky gallery, thumbnail switcher, variant pills, AJAX add to cart, accordion info sections, metafield support, related products
Wishlist SystemlocalStorage-based wishlist with heart buttons on all cards, badge counter in header, dedicated wishlist page
Account Dashboard3-tab dashboard: Orders, Addresses, Account Details. Works with Shopify New Customer Accounts
Cart PageAJAX quantity updates, remove with animation, coupon field, live totals, rewards banner
Contact PageIndustry type & needs pill selectors, contact info cards, Google Maps embed support, FAQ accordion
Privacy Policy PageSticky sidebar navigation, tab switcher (Privacy / PAIA), scroll spy, mobile collapsible TOC, fully dynamic blocks
Parallax FooterFixed-position footer that reveals as the page scrolls past
OS 2.0 CompatibleAll sections use JSON templates and Shopify block system for Theme Customizer editing

Browser Support

Chrome 90+, Firefox 88+, Safari 14+, Edge 90+. Mobile: iOS Safari 14+, Android Chrome 90+.

Installation

How to install the theme on your Shopify store

1

Upload the theme

In your Shopify Admin, go to Online Store → Themes → Add theme → Upload zip file. Select the apex-packaging-shopify-theme.zip file.

2

Preview before publishing

Click Preview on the uploaded theme to check everything looks correct before going live.

3

Create required pages

Go to Online Store → Pages and create the following pages. See the Required Pages section for full details.

4

Set up metafields

Go to Admin → Settings → Custom data → Products and create the product metafields. See the Metafields section for the full list.

5

Configure in Theme Customizer

Go to Online Store → Themes → Customize. Fill in your logo, navigation, contact details, and section content.

6

Publish

Once everything is configured, click Publish on the theme to make it live.

⚠️

This theme uses Shopify New Customer Accounts. Do not enable Legacy Customer Accounts or the account dashboard will not function correctly.

CSS Variables & Theming

How to change the theme colours and typography

The theme uses CSS custom properties (variables) for all colours. You can override them in assets/apex.css at the top of the file under :root.

:root {
  --c-bg:       #ffffff;   /* page background */
  --c-fg:       #0a0a0a;   /* primary text & dark elements */
  --c-muted:    #666666;   /* secondary text */
  --c-offwhite: #f7f5f2;   /* card backgrounds, subtle fills */
  --c-accent:   #2d5a27;   /* green accent (logo SVG) */
}

Typography

The theme uses two fonts loaded via Google Fonts in layout/theme.liquid:

FontUsage
Cormorant GaramondAll headings — h1, h2, section titles, hero text
System UI / -apple-systemBody text, labels, UI elements
💡

To change the heading font, find all instances of 'Cormorant Garamond', Georgia, serif in apex.css and replace with your preferred font. Make sure to update the Google Fonts link in layout/theme.liquid too.

Mega Menu Setup

The mega menu activates automatically for any top-level navigation item that has child links. The cards inside the dropdown are populated from those child links.

1

Create a navigation menu in Admin

Go to Online Store → Navigation → Main menu. Add top-level items. For items that should have a mega menu dropdown, add child links under them.

2

Upload dropdown images

In Theme Customizer → Header → Mega Menu Images, upload images for cards 1, 2, and 3. If no image is uploaded for a card, it will fall back to the collection image automatically.

3

Set collection images (auto-fallback)

If a dropdown card links to a collection URL (e.g. /collections/food-service), the theme will automatically use that collection's image if no manual image is set in step 2.

Wishlist Icon

The wishlist heart icon is built into the header automatically. It shows a red badge counter that updates when items are added. It links to /pages/wishlist.

Mobile Menu

On mobile, the navigation collapses into a slide-in drawer from the left. Navigation items with child links show an accordion that expands on tap. Parent items with children are not clickable — only their children are.

Related Products

Related products are automatically pulled from the collection that matches the current product's Product Type. If no matching collection is found, it falls back to collections/all.

Cart Page

AJAX cart with live totals

The cart page uses AJAX for all interactions — no page reloads required.

  • Quantity updates immediately via /cart/change.js
  • Items fade out and are removed via /cart/change.js with quantity 0
  • Subtotal, VAT, and Total update live after every change
  • Coupon code field included
  • Rewards banner configurable in Theme Customizer
ℹ️

VAT display is calculated at 15% for display purposes only. Shopify handles the actual tax calculation at checkout.

Account Dashboard

Customer account page with orders, addresses, and profile

The account dashboard lives at templates/customers/account.json and is only accessible to logged-in customers. Shopify automatically redirects logged-out visitors to the login page.

⚠️

This theme is built for New Customer Accounts (not Legacy). The login and signup pages are hosted by Shopify. You cannot and do not need to build custom login forms.

Tabs

TabContent
OrdersAll past orders with line items, prices, fulfilment status badges, and a "View order details" link
AddressesAll saved addresses in a grid. Add new address form (toggle), edit and delete per address
Account DetailsEdit name, email, phone number, and change password

Header Account Icon

Enable the account icon in Theme Customizer → Header → Show Account Icon. It links to routes.account_url — Shopify handles the redirect to login automatically if the customer is not logged in.

Setting up the contact form, info cards, and FAQ

Create a page in Admin

Go to Online Store → Pages → Add page. Title it "Contact Us". Set the handle to contact.

2

Set the template

In the right sidebar under Theme template, select page.contact.

3

Configure in Theme Customizer

Navigate to the Contact page in Theme Customizer. Fill in your address, phone, email, and operating hours. Add FAQ blocks using Add block → FAQ Item.

FAQ Blocks

Add as many FAQ items as you need via Add block → FAQ Item in Theme Customizer. Each block has a Question and Answer field. FAQs render as an accordion with smooth expand/collapse animation.

Form Fields

The contact form includes these fields by default: First Name, Last Name, Email, Phone, Company, Industry Type (pills), Needs (pills), and Message. All form submissions are sent to your Shopify store email via Shopify's built-in contact form handling.

Wishlist System

localStorage-based wishlist — no app required

How It Works

The wishlist is stored in the browser's localStorage under the key apex_wishlist. It stores product ID, title, price, URL, image, availability, and variant ID for each item.

⚠️

Because localStorage is browser-based, the wishlist is not synced across devices or accounts. It is cleared if the user clears their browser data. For cross-device sync, a third-party wishlist app would be needed.

Creating the Wishlist Page

1

Create a page in Admin

Go to Online Store → Pages → Add page. Title it "Wishlist". Set the handle to wishlist.

2

Set the template

Under Theme template, select page.wishlist.

Heart Buttons

Heart (♥) buttons appear on all product cards on the Collection page and on the main product image on the Product page. Clicking toggles the item in/out of the wishlist. The heart fills red when the item is wishlisted.

Header Badge

A red badge counter appears on the wishlist icon in the header when items are in the wishlist. It updates in real time as items are added or removed.

Wishlist Page Features

  • Shows all wishlisted products in a 4-column grid
  • Each card has an ✕ remove button and an Add to Cart button
  • Add to Cart uses AJAX — updates the header cart badge
  • Loading skeleton shown while JavaScript hydrates the page
  • Empty state shown with a Browse Products link when wishlist is empty

Footer

Parallax footer configuration

The footer uses a parallax effect — it is fixed to the bottom of the viewport and revealed as the page scrolls over it. This is achieved via CSS with the footer at position: fixed; z-index: 1 and the main content at position: relative; z-index: 2 with padding-bottom: 100vh.

Footer Logo

The footer has a dedicated Footer Logo image picker in Theme Customizer (Footer → Footer Logo). It falls back to the global store logo if not set, and then falls back to the shop name as text.

Setting Up Footer Links

Footer navigation links are pulled from Shopify's navigation system. Create a link list in Online Store → Navigation and assign it in Theme Customizer under the Footer section settings.

Product Tag Format

How to tag products for collection filters

The collection page filter sidebar automatically reads product tags in key:value format and groups them into filter categories. Tags without a colon are ignored by the filter system.

Format

material:Paper
material:PET Plastic
material:Bagasse
size:500ml
size:1L
size:2L

This would create two filter groups: Material (Paper, PET Plastic, Bagasse) and Size (500ml, 1L, 2L).

Special Tags

TagEffect
new or NewShows a "New" badge on the product card in the collection grid and product page
💡

Add tags to products in bulk via Admin → Products → select multiple → Add tags.
